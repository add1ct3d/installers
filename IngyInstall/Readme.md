# Ingenuity Information Scriptlet 
to download:
```
cd ~ && wget https://raw.githubusercontent.com/zaemliss/installers/master/IngyInstall/ingyinfo.sh -O ingyinfo.sh && chmod +x ingyinfo.sh
```

to execute the script:
```
cd ~
./ingyinfo.sh
```
